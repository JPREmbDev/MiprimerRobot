
# Blinkin LED

    En este caso configuraremos un proyecto para hacer parpadear un LED en la placa.


## TECNOLOGIAS USADAS

**SOFTWARE**

    - VSCODE
    - PLATFOMRIO
    - C/C++
    - GIT
    - CMAKE
    

**HARDWARE:**

    - ESP32
    - LED integrado en la placa GPIO_2
